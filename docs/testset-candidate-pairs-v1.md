# 第一版测试集候选配对表 V1

## 核心假设

- 这一版不是最终黄金测试集，而是 `第一轮候选样本池`。
- 每个方向先给出：
  - `1 个首选 JD`
  - `1 个备选 JD`
  - `1-2 个候选简历来源`
- 简历来源只作为能力结构素材，进入测试集前需要 `脱敏 + 必要改写`。

## 使用方式

- 先从这张表里挑出 `8` 组候选样本
- 再进入：
  - `case schema`
  - `rubric`
  - `人工标注`
- 如果某个方向的公开简历素材不够，就用现有来源池做匿名化改写补齐
