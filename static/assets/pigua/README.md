# 劈瓜帧动画素材

`design-preview-01.html` 会读取以下图片作为卡皮巴拉劈瓜帧动画：

- `frame-01.png`
- `frame-02.png`
- `frame-03.png`
- `frame-04.png`
- `frame-05.png`

当前内置素材来自 `tmp/imagegen_frames/640x480/`，尺寸为 `640x480`。

后续替换建议：

- 使用透明背景 PNG。
- 推荐尺寸为 `800x600` 或 `640x480`。
- 图片只画卡皮巴拉和瓜，不要把对白画死在图里。
- 当前 loading 动画只播放图片序列，不叠加对话框。

素材缺失或加载失败时，页面会显示占位卡片，不影响三步交互主流程。
